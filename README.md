## 🚀 Executando o Projeto

### Pré-requisitos

Antes de começar, você precisará ter instalado:

- Node.js
- npm ou yarn
- Expo Go (para execução em dispositivos móveis)

### Clonando o Repositório

```bash
git clone https://github.com/PamelaMBadermann/getting-started-react-native-expo.git

cd getting-started-react-native-expo
```

### Instalando Dependências

```bash
npm install
```

### Executando o Projeto

```bash
npx expo start
```

Após iniciar o servidor:

- Pressione `a` para abrir no Android Emulator.
- Pressione `i` para abrir no iOS Simulator (macOS).
- Ou escaneie o QR Code com o aplicativo Expo Go no celular.
