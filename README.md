# 📱 Estudos em React Native

## 🎯 Sobre o Projeto

Este projeto faz parte dos estudos desenvolvidos durante a graduação em Engenharia de Software e tem como propósito consolidar conhecimentos em desenvolvimento mobile utilizando React Native.

Mais do que uma aplicação final, este repositório funciona como um laboratório de estudos, onde novos conceitos, práticas e funcionalidades serão implementados conforme o avanço da minha jornada de aprendizado.

🚧 Projeto em desenvolvimento e evolução contínua.

## 🛠️ Tecnologias Utilizadas

- React Native
- Expo
- JavaScript
- Node.js
- Expo Go
- VS Code

## 🚀 Executando o Projeto

### Pré-requisitos

Antes de começar, você precisará ter instalado:

- Node.js
- npm ou yarn
- Expo Go (para execução em dispositivos móveis)

### Clonando o Repositório

```bash
git clone https://github.com/PamelaMBadermann/getting-started-react-native-expo.git

cd getting-started-react-native-expo
```

### Instalando Dependências

```bash
npm install
```

### Executando o Projeto

```bash
npx expo start
```

Após iniciar o servidor:

- Pressione `a` para abrir no Android Emulator.
- Pressione `i` para abrir no iOS Simulator (macOS).
- Ou escaneie o QR Code com o aplicativo Expo Go no celular.
